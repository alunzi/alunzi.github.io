前言

有朋友看到我的博客之后，说这个博客太牛了（指的是这个博客框架牛🤣🤣🤣，并不是说我的博客内容牛），还有个朋友说Hexo很优雅。确实，Hexo真的很牛又很优雅，很适合程序猿。其实之前就一直想分享一下Hexo的搭建，但就是懒，因为觉得内容还挺多的，当然，如果只是到可以用的程度那其实很简单，但每个人的追求不一样。

关于Hexo

Hexo是一款基于Node.js的静态博客框架，具有优秀的响应速度，同时支持Markdown，有着丰富的插件扩展，可玩性很高，只需要一条指令就可以方便的把生成的静态网页托管在Github或者其他平台。
